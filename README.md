# AI Resume Analyzer

An AI-based Resume Analyzer built using Python and Streamlit.
This application allows users to upload their resume and get insights such as detected skills, ATS score, and suggestions for improvement.

---

## 🚀 Features

* 📄 Upload Resume in PDF format
* 🔍 Extract text from resume
* 💡 Automatically detect skills
* 📊 Calculate ATS score (out of 100)
* 📌 Provide suggestions to improve resume

---

## 🛠️ Tech Stack

* Python
* Streamlit
* PyPDF2
* Pandas

## ▶️ How to Run the Project

1. Download or clone the repository

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the application:

```
streamlit run app.py
```

4. Open browser and go to:

```
http://localhost:8501
```
